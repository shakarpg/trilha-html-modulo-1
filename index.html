História dos computadores

A história dos computadores mostra como várias pessoas contribuíram para a evolução da tecnologia. Grandes nomes como Bill Gates (fundador da Microsoft) e Steve Jobs (fundador da Apple) ajudaram a popularizar o uso dos computadores pessoais. O HTML, por exemplo, nasceu para estruturar e organizar conteúdos na web.

"O computador é para todos" – frase que representa a visão de Steve Jobs.

O que são Clients?

O Client é a parte que o usuário final interage, ou seja, a camada visual da aplicação. Quando você abre um site no navegador, o que você vê é o lado do cliente (frontend), estruturado com HTML, CSS e JS.

O que são Servers?

O Servidor (Server) é o lado responsável por armazenar, processar e enviar as informações para os clientes. Enquanto o client é a parte visual, o servidor é considerado o backend, onde ficam os dados e regras de negócio.

Como é uma estrutura básica de um HTML?

Um documento HTML é composto por uma estrutura padrão:

  1. <!DOCTYPE html> → define o tipo de documento.
  2. <html> → tag raiz.
  3. <head> → contém informações como título, meta tags e links.
  4. <body> → onde fica o conteúdo visível da página.
Obs: Tags depreciadas, como font, já não são recomendadas.

Diferença entre listas ordenadas e não ordenadas

As listas ordenadas (<ol>) seguem uma sequência numérica ou alfabética:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Já as listas não ordenadas (<ul>) usam marcadores simples:

  • Item sem ordem
  • Outro item
  • Mais um item
